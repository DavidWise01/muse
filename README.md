**MUSE** is the art gallery of ROOT0 / David Lee Wise — generative engines, animated dimensional glyphs, and seal-works, gathered behind one front door. The landing page renders a **live thumbnail of every piece** (lazy-loaded), grouped into rooms.

**→ [davidwise01.github.io/muse](https://davidwise01.github.io/muse/)**

**18 works · 4 rooms.** Each piece is a self-contained HTML artwork in [`art/`](art) — most are live animated SVG/canvas, two are interactive engines (the d20 Creation Map in WebGL, the 54q 8K panning viewer).

## How it works

`index.html` is generated by [`build.py`](build.py) (stdlib only). It copies the source artworks into `art/` with clean slugs and lays them out as a gallery, each card carrying a lazy `<iframe>` thumbnail of the real piece (one heavy work, the 8K 54q viewer, shows a placeholder so the page stays light). To rebuild after adding a piece, edit the `ROOMS` map and run:

```bash
python build.py
```
